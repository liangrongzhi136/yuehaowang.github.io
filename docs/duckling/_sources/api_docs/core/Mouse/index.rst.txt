Mouse class
===========


A class enumerates mouse buttons, which is usually used for matching mouse button when processing mouse event.

**Base class:** ``object``

Attributes
----------


*static* :py:attr:`NONE`: ``NoneType`` **[Read Only]**

	- Default: None

	- Description: Not any mouse button.



*static* :py:attr:`LEFT_BUTTOM` **[Read Only]**

	- Default: GLUT_LEFT_BUTTON



*static* :py:attr:`RIGHT_BUTTOM` **[Read Only]**

	- Default: GLUT_RIGHT_BUTTON



*static* :py:attr:`MIDDLE_BUTTON` **[Read Only]**

	- Default: GLUT_MIDDLE_BUTTON
